Set an Audience

Targeting Audiences is what completes every campaign. Learn more about setting a custom audience with Twik's targeting KPIs.

Correct audience targeting is the secret to effective advertising. Twik offers more than your ordinary targeting by providing over 30 KPIs including demographics and interests, funnel and pre-funnel targeting*.

Form ideal audiences with advanced settings. You can set the following conditions:

  • Location – target users according to their geo-location
  • Age – define a preferred age range for your campaign
  • Gender – choose the gender you wish to target
  • Pages per session – limit the targeted audience to a number of minimal pages per session
  • Page views – set the audience according to number of page views
  • Interests – Twik is able to target audiences according to interests (such as learning, business, computing etc.)
  • Daily schedule – limit experiments for a certain time slot
  • Avg. session duration – set an audience according to the time a user spent on your website

Now it’s experimenting time. Read our article “Creating Experiments” to learn more about experiments.
